chip8as

This is an assembler for the CHIP-8 and SCHIP, utilizing libc8.

Table of Contents

  • Features
  • Usage

Features

  • Label support (mylabel:)
  • Support for strings (.DS), 16-bit data words (.DW), and data bytes (.DB)
  • Support for hex (0x or x prefixed), and binary (0b) integers

Usage

chip8as [-vV] [-o outputfile] src
  • -o sets an output file (default is a.c8).
  • -v prints diagnostic messages and the resulting hex-encoded bytecode to standard output.
  • -V prints the version number.

Example Code

JP main ; jump to main label
mysprite:
.DW 0x1234 ; sprite to draw
main:
ADD V0, 1 ; set x coordinate
ADD V1, 0xA ; set y coordinate
LD I, mysprite ; load sprite
DRW V0, V1, 1 ; draw sprite

Notes

  • Hex integers, besides V register identifiers, must be formatted with 0x, x, or $ prefixes.
  • Binary integers must be formatted with a 0b prefix.
  • Data bytes and strings are not padded. If a .DB or .DS contains an odd number of bytes, following bytecode will be offset (the first byte of following instructions will be at odd-numbered addresses).
  • Single-line comments start with ;
  • Commas are optional for instruction parameters.
  • This assembler is not case sensitive.
